Jan 25 2019 11:36 UTC
@mala57 great. i’m glad you finally got it working.
You have a few options with severities. 1) You can define your own severity levels 2) you can use a plugin to translate INFO => informational etc
see https://docs.alerta.io/en/latest/webui.html#webui and https://docs.alerta.io/en/latest/configuration.html#severity-settings to set your own severities
see https://docs.alerta.io/en/latest/gettingstarted/tutorial-3-plugins.html#tutorial-3 for how to write a python plugin that would do a translation. you would do this in the pre_receive() hook.
