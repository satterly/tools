@roboticpuppies The Docker image for Alerta has always (for a long time anyway) been configured to delete expired alerts. If you want to disable this feature then set ALERT_TIMEOUT=0 in the server configuration.

"@alerta/vue-authenticate": "github:alerta/vue-authenticate”,
