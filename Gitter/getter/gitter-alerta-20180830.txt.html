@isavcic this works. example here … https://github.com/alerta/prometheus-config#complex-example-using-correlation-without-resolve
however, it could have implications for rules that do not explicitly specify a “normal” rule so you never know when they resolve. ie. the tradeoff with making correlation work this way is that you need to define at least twice as many rules because you have to define a “normal” rule for every existing rule.
