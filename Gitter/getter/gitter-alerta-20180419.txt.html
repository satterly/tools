Nick Satterly
@satterly
Apr 19 2018 10:02 UTC
@jasonyates yes, if AUTH_REQUIRED is True in Alerta server settings then you need to use an API key. If it is not possible to set an header in the Grafana webhook POST configuration then just append an API key to the URL using api-key parameter http://… ?api-key=API_KEY_GOES_HERE.
See http://alerta.readthedocs.io/en/latest/authentication.html?highlight=api-key#api-keys

Nick Satterly
@satterly
Apr 19 2018 11:48 UTC
See http://alerta.readthedocs.io/en/latest/authorization.html?highlight=ADMIN_USER#configuration
basically, just add the list of admin usernames / email addresses to ADMIN_USERS server configuration. 
$ grep ADMIN_USERS ~/.alertad.conf
ADMIN_USERS=['admin@alerta.io', 'nfsatterly@gmail.com']
