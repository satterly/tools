Dec 09 2019 14:11 UTC
alertaConfig - is this something from Kubernetes? If so and if it is passed then to an Alerta Docker container as environment variables then it won't work because Alerta doesn't support environment variables to configure logging. You should pass your own Alerta config file inside the Alerta container. And this config file should contain correct settings for the logging subsystem (with quotes and all).
The list of supporter env variables: https://github.com/alerta/docker-alerta#environment-variables
How to pass a custom config file inside the alerta-web container: https://github.com/alerta/docker-alerta#configuration-files
^^^ @irom77
