hi @qed- if you’re using docker the alertmanager config for the webhook needs to change slightly. You need to add “/api” to the url so the receiverssection becomes something like …
receivers:
- name: "alerta"
  webhook_configs:
  - url: 'http://192.168.99.100:32770/api/webhooks/prometheus'
    send_resolved: true
Nick Satterly
@satterly
Mar 01 2016 20:36 UTC
@qed- and the scrape config for alerta will be different if you want to generate graphs or alerts from alerta metrics
scrape_configs:
  - job_name: 'alerta'
    metrics_path: /api/management/metrics
    target_groups:
      - targets: ['192.168.99.100:32770']
