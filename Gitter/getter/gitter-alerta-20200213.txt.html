Nick Satterly

Feb 13 17:46 UTC

If you need/want to build the image in a docker-compose file then you must define the Build Args in the build context like this ...

    build:
      context: .
      args:
        VCS_REF: ${VCS_REF}
        VERSION: ${VERSION}

… and set VCS_REF and VERSION as env variables on the build host.
See https://github.com/alerta/docker-alerta/blob/8a9105e30efd623735b8701302bcdab3d2d4f7b7/docker-compose.test.yml#L5-L9

But I still don’t understand why you think you need to build the image. Just use the image already available in the docker registry.
