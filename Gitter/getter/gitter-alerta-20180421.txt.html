@althahirie if you are logging in using alerta login then the Bearer token will expire after 14 days by default. you can change this to anything you want in the alerta server configuration file. https://github.com/alerta/alerta/blob/master/alerta/settings.py#L59
Alternatively, use an API Key. by default it does not expire for a full year. however, it is also a configurable setting.
