Nick Satterly

May 15 2019 21:53 UTC
I could make a change to allow LDAP groups to be used to auto-assign roles. ie. you would create Permission Lookups that map an LDAP group to a role (in addition to any roles that are directly assigned to that user).
But it would not being using the “groups” feature that exists in the web UI. That was added for “basicAuth” users where there is no other concept of a “group”. But LDAP has LDAP groups so to use both would be confusing.
