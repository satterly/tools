@richerve just to be clear, you asked if it was possible to use bearer tokens for the alerta API. I said it was and explained how. I would not actually recommend you integrate systems this way. That is what API keys are for. I would strongly recommend you always use API keys as they are more flexible (can be set in URL params for example) and can be revoked. Only use bearer tokens if you must -- tho I can't actually think why they would be mandated by some systems.
