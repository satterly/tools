Nick Satterly

alerta.app is the alerta server application. you have to install the entire server app just to get the python sdk if you do it that way. if you just want the python sdk use alertaclient.alert
alerta.app used to have a dependency on alertaclient but i pulled them apart just recently because of packaging and namespace issues.
so you might notice that alert.py and heartbeat.py in the two packages are identical.
