@satterly Alerts coming in from prometheus by default fall under Production environment... Is it possible to change it from Production to some other name like Unkown ?
Nick Satterly
@satterly
Feb 21 2017 20:00 UTC
@ramshankarjaiswal of course. see the example in the README https://github.com/alerta/prometheus-config#examples
global:
  external_labels:
    environment: Production
    service: Prometheus
    monitor: codelab
or use a server plugin to define it dynamically like this example https://github.com/alerta/alerta-contrib/blob/master/plugins/normalise/alerta_normalise.py
