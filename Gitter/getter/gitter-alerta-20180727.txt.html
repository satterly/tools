@n0guest Sure it is a little bit hacky. To integrate monit with Alerta I would create an python script using the Alerta python module and pass the arguments like this one: https://mmonit.com/wiki/MMonit/PushoverNotification that way you get more structured data. Then create an additional check using the program function https://mmonit.com/monit/documentation/monit.html#PROGRAM-STATUS-TEST this check would create an heartbeat like: https://github.com/alerta/python-alerta-client#python-sdk. That way you get an integration for alerts and heartbeats with only "one moving part". Using email would require logic to parse the email and then post it to Alerta, if your smtp goes down or similar you wouldn't get any notification or heads-up.
