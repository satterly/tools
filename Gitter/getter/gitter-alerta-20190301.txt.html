do you mean these instructions? https://www.digitalocean.com/community/tutorials/how-to-monitor-zabbix-alerts-with-alerta-on-centos-7
@te4336 ^
The only thing that needs to be installed on the Zabbix server is the zabbix_alerta.py script.
And then in the media config you point the zabbix server at alerta …
Type: Alerta
Send to: http://x.x.x.x:8080   => API hostname/IP and port of alerta server
When active: 1-7,00:00-24:00
Use if severity: (all)
Status: Enabled
follow the README here and let me know if you get stuck … https://github.com/alerta/zabbix-alerta
