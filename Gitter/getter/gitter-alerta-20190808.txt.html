@satterly Sorry, I was not able get back on oidc auth issue. I was doing some reading on how openid connect worked as I didnt know anything about it. That helped me figure out the issue, it was indeed mis-configuration. I had to set these variables in /etc/alertad.conf

OAUTH2_CLIENT_ID = '*************************'
OAUTH2_CLIENT_SECRET = "*****************************"
AZURE_TENANT = '***************************'
AUTH_PROVIDER = 'azure'

But major issue that bugged me the most was with alerta-webui. There was an issue with refreshing the page. Whenever I refresh the browser, I was getting a 404. Apparently, this happens in Vue. Had to search a lot to this info. The nginx config line that addressed that issue is posted below. It might help someone.

        location / {
                try_files $uri $uri/ /index.html;
                root /var/www/html;
        }

Still have the infinite loop issue only on Microsoft Edge. Firefox and chrome are fine. Although, it works just fine in Microsoft Edge in private mode.
