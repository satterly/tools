Nick Satterly
@satterly
May 16 2019 09:50 UTC
@Remy-Andreetti You don’t add LDAP users yourself. Users are added automatically when they first login. If you add them manually you’ll get database conflict errors when they try to log in first time.
Once they have logged in you can assign roles to them.
You can use LDAP groups to automatically assign customers to LDAP users using “customer lookups”.
If you want to use LDAP groups to automatically assign roles to users using “Permission lookups” (similar to customer lookups) I can add that. But you can’t do it right now.
