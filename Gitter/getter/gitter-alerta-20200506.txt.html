Hi @satterly, We have an origin that can configure the data in the json payload so we can use the standard /alert endpoint. The one thing that we can't change is the severity that it sends. It sends "down" and "trouble". We would like to map these to our standard "critical" and "warning". I have put these fields in the severity_map however we really want to "remap" these fields so they show up as critical/warning in the UI. Is this possible to do outside of a plugin or custom webhook?
