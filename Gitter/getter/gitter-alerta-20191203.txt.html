Dec 03 2019 11:34 UTC
Yes. Something like this: 
class TestPlugin(PluginBase):
    def status_change(self, alert, status, text, **kwargs):
        sid = alert.attributes.get("attr_name")
        if sid:
            return

        # some work with sid

        alert.attributes["attr_name"] = sid
But notice please that there are other hooks: pre_receive, post_receive and so on. They may be more useful.
