@kjetilmjos the AUTH_REQUIRED setting isn’t being picked up because of the spaces either side of the =
should be AUTH_REQUIRED=True
Also, ALERT_TIMEOUT and HEARTBEAT_TIMEOUT are not supported environment variables. if you want to set those settings you must use a configuration file.
see https://github.com/alerta/docker-alerta#environment-variables for list of support env vars
