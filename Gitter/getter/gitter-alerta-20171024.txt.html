you should be able to curl the endpoint and get the following error …

$ curl -XPOST http://localhost:8080/webhooks/grafana
{"code":400,"errors":null,"message":"no alerts in Grafana notification payload","status":"error”}

if you don’t get that error then you are not using the right URL or you are not POSTing.
