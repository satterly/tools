You can tweak any of these … QUERY_LIMIT , DEFAULT_PAGE_SIZE and HISTORY_LIMIT
Also the default page size in the web UI itself.
And refresh rate.
See https://docs.alerta.io/en/latest/configuration.html#api-settings for more info on those API server settings
