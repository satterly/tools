Nick Satterly

Sep 10 2019 18:32 UTC
riemann accepts HTTP events so yes, you could use a plugin to post alerts to Riemann.
actually, the website suggests using TCP inputs as there is less overhead. So you could use the logstash plugin for Alerta and send alerts via TCP to Riemann that way. See https://github.com/alerta/alerta-contrib/blob/master/plugins/logstash/alerta_logstash.py
