@moix beta.alerta.io is a github static site which does not support html5 mode redirects out-of-the-box. Ie. you need to do a lot of hacking around to get it work. Heroku allows you to create a wildcard redirect. See https://github.com/alerta/alerta-webui/blob/master/static.json#L6
