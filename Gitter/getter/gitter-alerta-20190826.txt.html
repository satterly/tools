@satterly Is there a way to change the menus on the left? I want to close some menu options for some of my users. Also. I can’t realy figurer out how the show panel works i.e. change the groupings.
Nick Satterly
@satterly
Aug 26 2019 17:55 UTC
Left hand menu options are displayed based on a users assigned permissions. If you don’t want them to see certain menu options then don’t assign those permissions.
For example, to see the “Heartbeat” menu a user must have at least “read:heartbeats” permissions. If they don’t have that perm the menu will be hidden.
See this code block for all menu-to-permission relationships … https://github.com/alerta/alerta-webui/blob/master/src/App.vue#L502
If the perm is “null” then no permission is required. This is true for Settings and Help.
