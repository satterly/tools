Nick Satterly
@satterly
Oct 18 2017 11:14 UTC
yes, that’s right. if you use the default “blackout” plugin you must have it listed in the PLUGINS setting. Then if you want alerts to appear in the web console even if they are within a blackout period you must set NOTIFICATION_BLACKOUT=True in the server config file.
In whatever notification plugins you want to use you should then check for an attribute called “notify” to see if it is set to False. if so, skip it.
does that make sense?
