Nick Satterly
@satterly
Aug 21 2019 12:06 UTC
Correlating two different alerts and only showing a higher level alert would require using an external service, such as Riemann or a complex event processor (CEP) like Esper (https://www.espertech.com/esper/)
403 = unauthorised (as different to unauthenticated) so you need to make sure any user or API key you are using has the correct permissions to create an alert
If you are using a non-default environment name then you need to set the ALLOWED_ENVIRONMENTS config setting.
all of this is documented online. in future i suggest you just google the error you receive and if you don’t get a hit then ask a question here. It will be quicker for you and less effort for me.
