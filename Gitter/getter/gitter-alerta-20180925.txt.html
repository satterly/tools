if you want to create API keys with limited scopes at the moment then you can use the command-line…

$ alerta key --help
Usage: alerta key [OPTIONS]

  Create or delete an API key.

Options:
  -u, --username TEXT  User (Admin only)
  --scope TEXT         List of permissions eg. admin:keys, write:alerts
  --duration SECONDS   Duration API key is valid
  --text TEXT          Description of API key use
  --customer STRING    Customer
  -D, --delete ID      Delete API key using ID or KEY
  -h, --help           Show this message and exit.

For example …

$ alerta key --text 'webhook only API key' --scope write:webhooks
EqtI2LEPb0BQ9TQk6p2cnnAgtXuz7hMAO4VuH5AG

The CLI can confirm the scopes …
