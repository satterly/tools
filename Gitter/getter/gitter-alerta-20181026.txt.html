@ireogunsina sounds like the API isn’t running at the host, port or path you have configured the client to connect to.
using a web browser or curl confirm that you can reach the API directly by going to the URL with a path /config. for example, https://alerta-api.herokuapp.com/config
this should return some JSON. if you don’t see the JSON then the API isn’t being served from where you think it is, or it isn’t running at all.
